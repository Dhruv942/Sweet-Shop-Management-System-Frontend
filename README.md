# TDD Frontend

A React application with User and CMS login functionality, set up with Jest for testing.

## Setup

Install dependencies:

```bash
npm install
```

## Development

Start the development server:

```bash
npm run dev
```

## Testing

Run tests:

```bash
npm test
```

Run tests in watch mode:

```bash
npm run test:watch
```

Run tests with coverage:

```bash
npm run test:coverage
```

## Build

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
